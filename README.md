# PRÁCTICA 5 INTELIGENCIA ARTIFICIAL
### Objetivo: 
#### El objetivo del programa es simular el algoritmo de Kruskal para construir árboles de mínimo y de máximo coste a partir de un grafo, mostrando paso a paso cómo se analizan las aristas, cuáles se aceptan o se descartan y cómo se forma el árbol final sin ciclos. Además, el programa genera una representación gráfica que permite visualizar de manera clara el crecimiento del árbol durante cada iteración, facilitando la comprensión del proceso completo del algoritmo.
